# FPS

This FPS game was designed by [Ray Chen](https://github.com/rayzchen) and was made with [Unity](https://unity.com).

## Supported platforms

Platform | Version
-------- | -------
Windows | 64-bit
WebGL | Microsoft Edge

## Controls

Controls are as follows:

Key | Usage
--- | -----
W | Forward
S | Backward
A | Left
D | Right
Q | Previous weapon
E | Next weapon
U | Use MedPack (+10 HP)
H | Use Potion (Restore HP)
Esc | Pause
